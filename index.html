Some Things I've Made

The Dictionary Project

The Dictionary App is a helpful tool for learning English that finds the meaning of any word and its pronunciation, points to a part of speech, gives examples of using the one in the context, also selects the necessary synonyms. And marvellous photos will remain in the learner's memory for a long time allowing to create a required association with an unfamiliar word. All in all, it makes the acquisition of English easier and faster.

The Weather Project

The app shows 7-days weather forecast including feels like temperature, wind speed, humidity and description (only for current day). Implemented converted units from celsius to faringeight and back, geolocation as well, which allows an user to see the weather forecast in his city in one click. The dynamic background with random photo of city makes view the weather forecast more interesting and puts familiar memory.
